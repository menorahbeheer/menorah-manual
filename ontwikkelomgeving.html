Dit hoofdstuk beschrijft hoe de CODESYS ontwikkelomgeving is opgezet, zodat de broncode van de verwarmingsregeling bekeken, aangepast en gedeployed kan worden.

ℹ CODESYS draait alleen op Windows

De CODESYS Development System IDE is alleen beschikbaar voor Windows. Mac-gebruikers draaien Windows via Parallels Desktop (Intel Mac) of Boot Camp. De huidige opstelling maakt gebruik van Parallels Desktop op een Intel MacBook Pro met Windows 11.

Installatie CODESYS

✓ Reeds geïnstalleerd

CODESYS V3.5 SP21 Patch 5 met WAGO Devices and Libraries 2.0.9.2 is reeds geïnstalleerd op de Windows omgeving. De onderstaande stappen zijn ter referentie voor een herinstallatie.

Stap 1: CODESYS bundle downloaden van WAGO

  1. Maak een gratis WAGO account aan op wago.com en upgrade naar Premium (gratis, bedrijfsgegevens van de kerk invullen)
  2. Ga naar de productpagina van de WAGO 750-8212
  3. Klik op het tabblad Downloads en zoek naar "Device Description CODESYS"
  4. Klik op de groene knop Bundle download — dit downloadt CODESYS én het WAGO device package samen (~2.8 GB)

Stap 2: CODESYS installeren

  1. Pak de gedownloade ZIP uit
  2. Dubbelklik op het CODESYS installatieprogramma (.exe)
  3. Kies installatietype Complete
  4. Accepteer de standaard installatielocatie (C:\Program Files\CODESYS 3.5.21.50\)
  5. Wacht tot de installatie voltooid is (kan 10-15 minuten duren)
  6. Herstart Windows na voltooiing

Stap 3: WAGO device package installeren

  1. Start de CODESYS Installer via het Startmenu (rechtermuisknop → Uitvoeren als administrator)
  2. Klik op Change bij de CODESYS 3.5 SP21 installatie
  3. Klik op Install File(s) rechtsbovenin
  4. Navigeer naar het WAGO .package bestand en selecteer het
  5. Bevestig de installatie — sluit CODESYS indien open
  6. Controleer na installatie: WAGO Devices and Libraries 2.0.9.2 staat in de Installed lijst

Projectarchief ophalen bij Van Niel

Voor toegang tot de broncode dient het volledige projectarchief te worden opgevraagd bij Van Niel Installatietechniek. Het archief bevat de broncode, libraries en apparaatconfiguratie.

Verzoek aan installateur

Vraag Van Niel om het project te exporteren via Project → Archive → Save archive in CODESYS. Het resulterende .projectarchive bestand kan per e-mail worden aangeleverd (typisch 50 KB – 2 MB).

Vraag tegelijk om:

  • De huidige firmware versie van de PLC (zichtbaar via het WAGO webinterface)
  • Een overzicht van de I/O modules op de rail

Project openen in CODESYS

  1. Start CODESYS V3.5 SP21 via het Startmenu
  2. Ga naar File → Open → Project Archive...
  3. Navigeer naar het ontvangen .projectarchive bestand
  4. CODESYS pakt het archief uit en opent het project
  5. In de linker Devices-boom zie je de projectstructuur: Device (PLC), PLC Logic, Application, PLC_PRG, etc.

Verbinden met de WAGO PLC

⚠ Alleen verbinden als je bewust wijzigingen wilt doorvoeren

Verbinden met de productie PLC en een programma downloaden overschrijft de draaiende code. Zorg altijd voor een back-up van het huidige project voordat je iets deployt. Test wijzigingen eerst in de CODESYS simulatie (Control Win V3).

  1. Zorg dat je laptop op hetzelfde netwerk zit als de WAGO PLC (of via VPN)
  2. Ga naar Online → Login (Alt+F8)
  3. Klik Yes bij de netwerkscan — CODESYS zoekt naar beschikbare PLCs
  4. Selecteer de WAGO PFC200 in de lijst
  5. Log in met de admin credentials
  6. Na verbinding zie je in de Devices-boom: Device [connected]

Lokale simulatie (zonder PLC)

Wijzigingen kunnen eerst worden getest in de CODESYS software-simulatie voordat ze naar de echte PLC worden gedeployed.

  1. Start CODESYS Control Win V3 x64 SysTray via de Windows taakbalk
  2. Klik Start PLC — bevestig de beveiligingswaarschuwing met OK
  3. In CODESYS: ga naar Online → Login
  4. CODESYS vindt de lokale simulatie-PLC automatisch
  5. Activeer user management als gevraagd (Yes) en maak een admin gebruiker aan
  6. Na login: Application staat op [stop] — druk F5 of klik de Start knop om het programma te starten
